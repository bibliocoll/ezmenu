GO

Welcome to "GO", the EZProxy installation provided to you by your library.

How-to:

  • The "GO"-Button (bottom left of this page) contains a menu with links to all online resources available via this proxy
  • The Button and menu will be injected into all pages you visit from links in the menu, unless they are marked with the -icon, in which case the page will open in a new tab
  • If you encounter the -icon on a resource that is not freely available to the public, please let us know
  • If you think the collections we curated are missing something, please also let us know
  • Please use the logout button in the menu to end your session

Troubleshooting tips: If you are using privacy enhancing browser plug-ins, you may have to allow javascript and iframes from http(s)://go.YOUR-DOMAIN.TLD on all subdomains ("*.go.YOUR-DOMAIN.TLD") to get button and menu to show up on proxied pages.
